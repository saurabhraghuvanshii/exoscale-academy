![Congratulations](./congratulations.jpeg)

You've just completed this workshop. We hope it gave you a good overview of the Exoscale offerings.

Do not forget to perform the cleanup tasks to remove the resources created during this workshop:
- delete the SOS bucket
- delete the SKS cluster
- delete the NLB
- Disable the managed Vault in the Marketplace

{{< hextra/callout type="warning" >}}
Since the NLB was created during the installation of the Ingress Controller, it is not automatically removed when deleting the cluster
{{< /hextra/callout >}}

Explore the [Exoscale documentation](https://community.exoscale.com/documentation/) to learn about the global offering. You may also consider joining the [Exoscale Academy](https://www.exoscale.com/academy/) to learn from Exoscale cloud experts.
