# Challenges

Test your troubleshooting skills with these hands-on challenges. Each challenge intentionally breaks something in your cluster, simulating real-world issues.

## How it Works

1. Run a challenge setup script to introduce a problem into your cluster
2. Diagnose the problem 
3. Fix it

{{< hextra/callout type="warning" >}}
**Prerequisites**: These challenges require a cluster created with the Multipass environment as defined in the [cluster creation](../creation) section.
{{< /hextra/callout >}}
